A λ crawler

Welcome, good visitor! You find yourself on the page of Hephaestos. Hephaestos is a crawler library/client with which can perform a variety of tasks, such as:

  • download entire sites,
  • download all images/links on a site,
  • download webcomics,
  • download YouTube videos (shhhh!)
  • run your own crawler
The program is basically a library, intended for programmers, but it also includes a CLI for easy use. That's enough chit-chat.

Features

Hephaestos is a crawler-writing framework that comes with a couple of default crawlers. A crawler, for our purposes, is a function that takes a website and generates a bunch of results, together with URLs that should also be crawled. Once you have that, Hephaestos takes care of everything else: saving, error-reporting, re-trying, post-processing. While you can just mirror an entire website in this way, it is also possible to just download, say, a series of images, text, XML, or videos from it.

An example, in the CLI:

$ ./Hephaestos
> :crawler pennyArcade
> Enter direction (Forward/Backward): Backward
> Enter number of items: 10
...
Job done!

And thus, the last ten issues of the Penny Arace webcomic have been downloaded. A second, site-generic example:

$ ./Hephaestos
> :crawler allImages
> Enter URL: http://www.domain.com/kittenPictures
...
Job done!

This series of commands downloads all linked images from the fictional http://www.domain.com/kittenPictures.

How complicated is it to use?

I am a very, very lazy many and thus, you can rest assured that this tool is optimized for the absolute minimum of user effort.

Requirements & Installation

You will need Haskell (GHC version 7.6 or newer recommended). The platform can be gotten from https://www.haskell.org/platform/. Once it is installed, type the following: 
$ cd directory_into_which_you_extracted_hephaestos
$ cabal install hephaestos.cabal
$ runhaskell Setup.hs install

This installs the library and creates the executable (called Hephaestos).
